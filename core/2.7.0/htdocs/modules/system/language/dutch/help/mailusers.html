Help: E-mailgebruikers

Beschrijving

XOOPS heeft een ingebouwd berichtensysteem waarmee u een bericht kunt sturen naar uw gehele geregistreerde gebruikersbestand of subgroepen daarvan, of door datumfilters te gebruiken. Dit kan de vorm hebben van een privébericht (d.w.z. via het ingebouwde berichtensysteem van XOOPS) of via een e-mail. Er is een scala aan opties beschikbaar voor het filteren van subgroepen van gebruikers uit uw lidmaatschap.

U kunt ook vooraf gedefinieerde tags gebruiken waarmee u uw bericht kunt personaliseren.

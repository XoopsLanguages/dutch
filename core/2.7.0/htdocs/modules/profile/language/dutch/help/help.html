BESCHRIJVING

De Profielmodule is bedoeld voor het beheren van aangepaste gebruikersprofielvelden.

INSTALLEREN/VERWIJDEREN

Er zijn geen speciale maatregelen nodig, volg het standaard installatieproces – pak de map /profile uit in de map ../modules. Installeer de module via Beheer -> Systeemmodule -> Modules.

Gedetailleerde instructies voor het installeren van modules zijn beschikbaar in de XOOPS-bedieningshandleiding

GEBRUIKSAANWIJZING

Om deze module in te stellen, heeft u het volgende nodig:

  • Configureer uw voorkeuren voor de module (zie ‘Voorkeuren’) en optioneel het Profielblok als u van plan bent deze te gebruiken (zie ‘Blokken’).
  • Bewerk bestaande categorieën of voeg nieuwe toe.
  • Bewerk bestaande velden of voeg nieuwe toe. Hier kunt u opgeven welke velden zichtbaar zijn in welke categorie, en of ze zichtbaar zijn tijdens de gebruikersregistratie.
  • Definieer de volgorde van de registratiestappen.
  • En tot slot kunt u machtigingen instellen voor individuele velden: welke velden kunnen worden bewerkt en welke doorzoekbaar zijn.

Antispammaatregelen

Om spamregistraties te minimaliseren, doet u het volgende:

  • ga naar de Protector-module in Admin, ga naar Voorkeuren en vervolgens onderaan naar deze optie: "Stop forumspam" Controleert POST-gegevens op spammers die zijn geregistreerd in de database www.stopforumspam.com.
    Stel dit in op "Ban the IP (no limit)"

  • in /class/captcha/config.php, zorg ervoor dat de modus is ingesteld als "tekst":

    return $config = array(
    'uitgeschakeld' => false, // CAPTCHA uitschakelen
    'mode' => 'text', // standaardmodus, je kunt kiezen uit 'text', 'image', 'recaptcha' (api-sleutel vereist)
    'naam' => 'xoopscaptcha', // captcha-naam
    'skipmember' => true, // CAPTCHA-controle voor leden overslaan
    'maxattempts' => 10, // Maximaal aantal pogingen voor elke sessie
    );

  • Ga in de Profielmodule naar Beheerder en in de Basisstap naar:
    /modules/profile/admin/step.php?id=1
    zet "Opslaan na stop" op "Nee"


  • Stel in Profielvoorkeuren de optie "Gebruik Captcha na de tweede registratiestap" in op "Ja" (dit is de standaardinstelling)
